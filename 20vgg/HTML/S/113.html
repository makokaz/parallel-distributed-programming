<a name="details" id="details"></a><h2 class="groupheader">Detailed Description</h2>
<div class="textblock"><h3>template&lt;idx_t maxB, idx_t C, idx_t H, idx_t W&gt;<br />
struct array4&lt; maxB, C, H, W &gt;</h3>

<p>4D array </p>
<dl class="params"><dt>Parameters</dt><dd>
  <table class="params">
    <tr><td class="paramname">(maxB)</td><td>the maximum number of rows (elements along the first dimension) </td></tr>
    <tr><td class="paramname">(C)</td><td>the number of elements along the second dimension </td></tr>
    <tr><td class="paramname">(H)</td><td>the number of elements along the third dimension </td></tr>
    <tr><td class="paramname">(W)</td><td>the number of elements along the fourth dimension</td></tr>
  </table>
  </dd>
</dl>
<p>this is essentially BxCxHxW array of reals where B can be a runtime parameter &lt;= maxB. throughout the <a class="el" href="structVGG.html" title="VGG network. ">VGG</a> network, is is used to represent a mini-batch of images (B images, each image of which has C channels, each channel of which has HxW pixels. </p>
