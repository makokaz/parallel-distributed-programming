<a name="details" id="details"></a><h2 class="groupheader">Detailed Description</h2>
<div class="textblock"><h3>template&lt;idx_t maxB, idx_t nC&gt;<br />
struct SoftmaxCrossEntropy&lt; maxB, nC &gt;</h3>

<p>dropout layer </p>
<dl class="params"><dt>Parameters</dt><dd>
  <table class="params">
    <tr><td class="paramname">(maxB)</td><td>the maximum number of images (batch size) </td></tr>
    <tr><td class="paramname">(nC)</td><td>number of classes (10)</td></tr>
  </table>
  </dd>
</dl>
<p>input is essentially a two dimensional vector describing the score for each image and each class. the score for image i and a class c is a likelihood that the image i belongs to the class c. based on this matrix, it first converts the vector for each image to the probability vector using the softmax function. it then compares the probability score with the true label and calculate the loss using the cross entropy function. </p>
