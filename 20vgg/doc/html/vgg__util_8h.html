◆ algo_is_gpu()

int algo_is_gpu ( const char *  s,
algo_t  a 
)

return 1 if the algorithm name (s) or its enum value (a) is a gpu algorithm

when you add your algorithm, you may need to change this function so that it correctly recognizes whether it is a gpu algorithm or not currently, it considers all and only strings starting with "gpu" to be a gpu algorithm. for a gpu algorithm, the program transfers initial weights and training data to gpu. weights stay on GPU until the program finishes.
