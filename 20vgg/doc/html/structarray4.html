Detailed Description

template<idx_t maxB, idx_t C, idx_t H, idx_t W>
struct array4< maxB, C, H, W >

4D array

Parameters
(maxB)the maximum number of rows (elements along the first dimension)
(C)the number of elements along the second dimension
(H)the number of elements along the third dimension
(W)the number of elements along the fourth dimension

this is essentially BxCxHxW array of reals where B can be a runtime parameter <= maxB. throughout the VGG network, is is used to represent a mini-batch of images (B images, each image of which has C channels, each channel of which has HxW pixels.
