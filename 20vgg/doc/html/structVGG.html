Detailed Description

template<idx_t maxB, idx_t C0, idx_t H, idx_t W, idx_t K, idx_t S, idx_t C1, idx_t nC>
struct VGG< maxB, C0, H, W, K, S, C1, nC >

VGG network.

Parameters
(maxB)maximum batch size it can accommodate (64)
(C0)the number of channels in an input image (3)
(H)height of an input image (32)
(W)width of an input image (32)
(K)convolution kernel size (1). filter array has (2K+1)*(2K+1) elems)
(S)shrink factor of pooling layers (2)
(C1)number of channels in the first hidden layer (64). those in the following layers are C1/2, C2/4, C3/8, ..., 1
(nC)number of classes (10)

calc the gradient of loss wrt the input (x)

Parameters
(gy)gradient of loss with respect to the output

calc the gradient of loss wrt the input. along the way, it also calculates the gradient of loss wrt weights for all sublayers that have weights. since this is the entire network, gy is actually a vector whose components are all 1. (loss = sum of losses of each data).
