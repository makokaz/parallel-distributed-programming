Detailed Description

template<idx_t maxB, idx_t nC>
struct SoftmaxCrossEntropy< maxB, nC >

dropout layer

Parameters
(maxB)the maximum number of images (batch size)
(nC)number of classes (10)

input is essentially a two dimensional vector describing the score for each image and each class. the score for image i and a class c is a likelihood that the image i belongs to the class c. based on this matrix, it first converts the vector for each image to the probability vector using the softmax function. it then compares the probability score with the true label and calculate the loss using the cross entropy function.

calc the gradient of loss wrt the input (x)

Parameters
(gy)gradient of loss with respect to the output

calc the gradient of loss wrt the input. along the way, it also calculates the gradient of loss wrt weights for all sublayers that have weights. since this is the entire network, gy is actually a vector whose components are all 1. (loss = sum of losses of each data).

compute log(softmax(x))

Parameters
(x)a matrix

for 1D vector x x = (x_0, ..., x_{n-1}), 

                  (exp(x_0)     / Σ_j exp(x_j))

logsoftmax(x)_i = log (exp(x_1) / Σ_j exp(x_j)) ( ... / Σ_j exp(x_j)) (exp(x_{n-1}) / Σ_j exp(x_j))

the input to this function is essentially a two dimensional matrix (4D array whose last two axes have only one element), which is simply a set of vectors.
